=== TurnSocial Toolbar Plugin ===
Contributors: zaius
Stable tag: 0.1.2

Allows you to easily display your TurnSocial toolbar on your wordpress blog.
Don't have a toolbar? [Create one for free!](http://turnsocial.com)

== Installation ==

1. Extract the zip file to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the 'TurnSocial Configuration' in your plugin options menu and follow 
the instructions.
